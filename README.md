# 🎨 UIKit - Framework CSS Educativo

**Un proyecto educativo completo para aprender cómo funciona UIKit**, un framework CSS moderno y flexible para crear interfaces web atractivas y responsivas.

> **¡Bienvenido!** Este proyecto contiene documentación, ejemplos prácticos y recursos completos para dominar UIKit desde cero.

## 🔗 CDN - Listo para Copiar

```html
<!-- Hoja de estilos -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/uikit@3/dist/css/uikit.min.css">

<!-- JavaScript -->
<script src="https://cdn.jsdelivr.net/npm/uikit@3/dist/js/uikit.min.js"></script>

<!-- Iconos -->
<script src="https://cdn.jsdelivr.net/npm/uikit@3/dist/js/uikit-icons.min.js"></script>
```
