# Ejemplos Prácticos de UIKit

En esta carpeta encontrarás ejemplos completos de diferentes tipos de proyectos usando UIKit.

## Lista de Ejemplos

### 1. **Blog con UIKit** (`01-blog.html`)
Un blog completo con:
- Grid responsivo para artículos
- Sidebar con búsqueda y categorías
- Cards para artículos con imágenes
- Sistema de etiquetas
- Paginación

**Conceptos**: Grid, Cards, Utilidades de espaciado, Badges

---

### 2. **Dashboard** (`02-dashboard.html`)
Un dashboard administrativo con:
- Navbar sticky
- Tarjetas de estadísticas con iconos
- Gráficos y tablas
- Actividad reciente
- Tabla de transacciones con estado

**Conceptos**: Flexbox, Cards, Iconos, Tablas, Badges

---

### 3. **Formulario de Contacto** (`03-formulario.html`)
Formulario completo y validado con:
- Campos de texto, email, teléfono
- Textarea para mensajes
- Select y opciones
- Checkboxes y radios
- Validación en tiempo real
- Mensaje de confirmación

**Conceptos**: Formularios, Inputs, Validación CSS, Alerts

---

### 4. **Landing Page** (`04-landing-page.html`)
Página de ventas profesional con:
- Hero section con CTA
- Sección de características
- Tabla de precios con plan destacado
- Testimonios
- Sección de contacto
- Footer

**Conceptos**: Hero section, Grid, Cards especiales, Gradientes, Botones personalizados

---

## Cómo Usar Estos Ejemplos

1. **Abre cualquier archivo HTML** en tu navegador
2. **Inspecciona el código** para ver cómo está estructurado
3. **Modifica y experimenta** - Los ejemplos son para aprender!
4. **Copia patrones** que te gusten para tus proyectos

## Tips Importantes

- Todos los ejemplos usan **UIKit desde CDN**, no necesitas instalación
- Usa la **Consola del Navegador** (F12) para debuguear
- Prueba en **diferentes tamaños de pantalla** (responsive)
- Inspecciona las **clases de UIKit** para entender mejor cómo funcionan
- Compara los **ejemplos entre sí** para ver diferentes patrones

## Estructura Típica

Todos los ejemplos siguen esta estructura básica:

```html
<!DOCTYPE html>
<html>
<head>
    <!-- Meta tags -->
    <!-- Enlace a UIKit CSS -->
    <!-- Estilos personalizados -->
</head>
<body>
    <!-- Navbar/Header -->
    <!-- Contenido principal -->
    <!-- Footer -->
    
    <!-- Scripts UIKit -->
    <!-- JavaScript personalizado -->
</body>
</html>
```
